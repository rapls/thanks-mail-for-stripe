# Thanks Mail for Stripe

Stripe決済の完了時に、サンクスメールを自動送信できる WordPress プラグインです。Payment Link ごとに異なるテンプレートを設定でき、ライセンスキー配信にも対応。

📖 **詳しい解説記事**: [Thanks Mail for Stripe 開発者が解説｜なぜ作ったか・セットアップ・ハマりポイントまで](https://raplsworks.com/thanks-mail-for-stripe/)

## Features

- Stripe Webhookを受けて自動メール送信
- 最大10種類のテンプレート（Payment Link ID で切り替え）
- ライセンスキー配信にも対応
- 日本語・英語の双方に対応

## Installation

### WordPress.org から（推奨）

WordPress管理画面 → プラグイン → 新規追加 → 「Thanks Mail for Stripe」で検索

### GitHub から

Releases から最新版の ZIP をダウンロード。

## Requirements

- WordPress 5.0以上
- PHP 7.4以上
- Stripe アカウント

## Use Cases

- デジタル商品販売後のライセンスキー自動配信
- 購入後のフォローメール
- 商品別の案内メール送り分け
